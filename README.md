# Ionic Test Project

Trying out ionic with a sample app. Uses jade

## Dependencies
- `node, bower, gulp`
- `npm install -g cordova ionic`

## Running tasks
### Run app on server
- `gulp serve`


### Run app on ios
- `ionic platform add ios`
- `ionic build ios`
- `ionic emulate ios`
